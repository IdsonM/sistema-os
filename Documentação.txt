Sistema de Gerenciamento de Ordens de Serviço
Autor: Idson Magalhaes Malato
Data: 29/05/2026

1. OBJETIVO
Este documento descreve os requisitos funcionais de um sistema para gerenciamento de Ordens de Serviço (OS), permitindo o cadastro, edição, acompanhamento e comunicação com clientes.

2. ESCOPO DO SISTEMA
O sistema permitirá:

Autenticação de usuários
Cadastro de ordens de serviço
Consulta e pesquisa de OS
Atualização de status
Geração de relatórios/exportação
Integração com WhatsApp


3. TELA INICIAL
A tela inicial deverá conter:
3.1 Autenticação

Campo Login
Campo Senha
Link “Esqueci minha senha”

Redireciona para tela de redefinição de senha



3.2 Contato

Botão “Contatar Administrador”

Exibe informações de contato do administrador (telefone, e-mail ou WhatsApp)




4. TELA PRINCIPAL (APÓS LOGIN)
4.1 Cabeçalho


Lado esquerdo: Nome da empresa


Lado direito: Tipo de usuário logado:

Administrador
Usuário final
Cliente



Botão “Sair” (Logout)



5. PESQUISA DE ORDENS DE SERVIÇO

Campo de pesquisa por número da OS
Retorno rápido dos registros correspondentes


6. CADASTRO DE ORDEM DE SERVIÇO
Localizado na lateral esquerda da tela.
Campos obrigatórios:

Nome do cliente
Equipamento
Descrição / defeito
Data de entrada
Orçamento
Contato

Ação:

Botão “Gerar Ordem de Serviço”


7. LISTAGEM DE ORDENS DE SERVIÇO
Localizada na área central da tela.
7.1 Exibição

Listagem em ordem de cadastro
Visualização de todos os dados informados

7.2 Ações por registro
Cada OS deve apresentar:

Editar
Excluir
Imprimir
Status (editável):

Em análise
Em reparo
Concluído



7.3 Exportação

Botão para Exportar para Excel (.xlsx)


8. INTEGRAÇÃO COM WHATSAPP
O sistema deve permitir:

Recebimento de solicitações de clientes
Comunicação direta com o cliente
Envio de informações relacionadas à OS


9. PERFIS DE USUÁRIO
Administrador

Acesso total ao sistema

Usuário Final

Cadastro e consulta de OS

Cliente

Consulta de ordens de serviço


10. REQUISITOS GERAIS

Interface intuitiva e de fácil uso
Sistema responsivo (desktop e mobile)
Segurança na autenticação
Armazenamento seguro dos dados
